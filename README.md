# Simple Card Game Logic

A Python-based foundation for a card game (Blackjack-style) focusing on turn initialization and value mapping.

## 🃏 Current Features
* **Turn Selection:** Simple input logic to decide if the player or the opponent draws first.
* **Card Value Mapping:** A comprehensive dictionary assigning numeric values to standard playing card ranks (including face cards and Aces).

### Prerequisites
* Python 3.10 or higher (required for `match-case` statements).
